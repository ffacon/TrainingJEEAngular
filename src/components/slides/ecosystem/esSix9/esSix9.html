Ecosystem ES6 Introduction - Sets And Maps

Sets and Maps didn't exist in JavaScript before ES6, which was a pain each time you need this kind of functionality. Happily, they are now available!

You can use any object for keys, but equality of keys is checked through the '===' operator, which can be error prone : ['hello'] === ['hello'] //false

//Map class Person{ constructor(id, firstName, lastName, age){ //... this.id= id; this.friends= new Map(); } addFriend(friend){ this.friends.set(friend.id, friend); } getFriendById(id){ this.friends.get(id); } } //Set let friend= new Person('a', 'b'); let friends= new Set(); friends.add(friend); friends.has(friend); // true
