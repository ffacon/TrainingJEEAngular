Ecosystem Typescript - Definition Files

Typescript files generally have the .ts extension. However, some files have the .d.ts one, which means they contain type definitions (no implementations). This is particularly usefull when an existing javascript library/framework wants to be available for typecript projects without the need to rewrite the library in typescript.

In such a case, the compiler has to know where are these definition files:

  • Reference them in the file: 
    ///path/to/file.d.ts
  • Put them on the root of the project

In order to make this process industrializable, you can use tools such as tsd or typings with DefinitelyTyped
