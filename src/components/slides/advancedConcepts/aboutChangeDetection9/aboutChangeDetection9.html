Advanced Concepts Change Detection - Note

What the point with Observables

  • They were not mandatory to illustrate our previous sample
  • But it's not a matter of observables
  • We could use an array as well, as long as the array reference doesn't change
  • The main concept is mutability of objects
  • When you modify an object, by adding it a field, the object reference remains the same
  • Then the OnPush strategy won't work, unless you use the markForCheck() method
  • An other way to deal with this is to use immutable objects for input parameters: this can be provided by libraries which will assure that when an object is modified, you get a new reference of this object
