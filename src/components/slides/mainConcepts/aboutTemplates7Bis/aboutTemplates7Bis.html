Main Concepts About Templates - Property Binding - Declarations

  • In previous examples, we used the @Input annotation to declare a parameter for a component
  • And you can use the inputs attribute in the @Component annotation as well
import {Component} from '@angular/core'; import {BrowserModule} from '@angular/platform-browser'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'app', template: '<sub-cmp name="john" ></sub-cmp>' }) class App {} export const ModuleDefs= { bootstrap: [App], declarations:[ App, SubCmp], imports: [BrowserModule] } import {Component} from '@angular/core'; @Component({ selector: 'sub-cmp', template: ` <h1> Name : {{name}}</h1> `, inputs: ['name'] }) export class SubCmp { name: string; }
