Main Concepts Classes Are Everywhere - Components

  • A component is also an ES6 class, but annotated which @Component
  • The @Component annotation allows to add some metadata, such as the template/templateUrl ones
  • The template can use the public methods attributes declared in the class

Below is a sample of Component

@Component({ selector: 'hello-world', template:'<span>Hello {{name}} </span>', inputs: ['name'] }) class Redisplay{ public name: string='world'; constructor(){} }
