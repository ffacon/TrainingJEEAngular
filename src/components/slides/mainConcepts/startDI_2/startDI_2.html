Main Concepts Dependency Injection - Introduction

Whatever A and B are, if I want to inject A in B

  • Without DI:
    • Create A: var a= new A();
    • Attach A to B: b.setA(a);
  • With DI:
    • Tell the framework that B depends on A, and let the framework control this dependency: 
      // configuration feature somewhere
// ! THIS IS NOT ANGULAR 
config.declareDeps(A, [B]);
