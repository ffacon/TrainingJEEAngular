Ecosystem ES6 Introduction - Rest Operator

The rest operator ( ... ) is more clear and let us work with a specified parameter name. It can also be used when destructuring data.

let getSum= function(...toAdd){ let sum= 0; for (int i= 0; i < toAdd.length; i++){ sum+= toAdd[i]; } }; let getSumButFirst= function(...toAdd){ let [first, ...next] = toAdd; for (int i= 0; i < next.length; i++){ sum+= next[i]; } };
