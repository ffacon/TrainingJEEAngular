Main Concepts PW - Templates

The aim of this PW is to let you manipulate simple templates:

  • To initialize your app, if not already done, run:
    • npm install
    • If you don't have a global gulp executable : npm install -g gulp
    • If you don't have a global tsd executable : npm install -g tsd
    • tsd install
    • npm run jspm
  • To run your server only: npm run serve
  • To compile your ts only: npm run compile
  • To do both: npm run dev (as long as you don't create new ts files, each modification will be compiled automatically, if you need to create a new typescript file run the compile target manually)
  • Access your application here
  • Have a look at the app folder structure to have an idea of where are the things you'll have to implement. Ask if you have any question!
  • You will have to use the {{expression}} in order to bind a value from the component into the template ( there is an example slide 69 if you have some doubts).
  • Browse to bookDetails.html
  • Complete the TODO_BASICS comments in the template (displays informations about the book)
  • Go to this url to check your devs: http://localhost:8081/#/book/1

If book is undefined or null, {{book?.xxx}} will do the job
