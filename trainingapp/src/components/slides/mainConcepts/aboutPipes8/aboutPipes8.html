Main Concepts Pipes - Custom Ones

  • And as for services and directives, you can create your own custom Pipes
  • Use the @Pipe annotation
  • Implement the PipeTransform interface
  • Declare your pipe in your module declarations field
  • Below is an example of "Hello World" Pipe
import {Component, Pipe, PipeTransform} from '@angular/core'; import {BrowserModule} from '@angular/platform-browser'; @Pipe({ name:'helloWorld' }) class HelloWorldPipe implements PipeTransform{ transform(value: string, args:any[] = null){ return 'Hello ' + value; } } @Component({ selector: 'app', template: ` <p> {{'John' | helloWorld }}</p> ` }) export class App {} export const ModuleDefs= { bootstrap: [App], imports: [BrowserModule], declarations: [App, HelloWorldPipe] }
