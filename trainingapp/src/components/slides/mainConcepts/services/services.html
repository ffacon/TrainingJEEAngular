Main Concepts Classes Are Everywhere - About Services - Existing ones

  • Services can be used everywhere in your app
    • In other services
    • In components/directives
    • In pipes
  • Dependency Injection details come in a few slides, for now consider than to use a service, you have to:
    • Either import the module which owns this service if it comes from an other module import {NgModule} from '@angular/core'; import {ExternalModule} from '/path/to/module.ts'; @NgModule({ imports:[ ExternalModule //This makes all services exported by ExternalModule available in our module ], ... bootstrap: [AppComponent] }) export class AppModule
    • Or declare it in your own module import {NgModule} from '@angular/core'; import {MyCustomService} from '/services/myCustomService.ts'; @NgModule({ imports:[...], providers:[MyCustomService], ... bootstrap: [AppComponent] }) export class AppModule
    • And then use it into the constructor of your directives/services to retrieve it at some point export class MyAppComponent{ constructor(private service: MyService){} //... }
