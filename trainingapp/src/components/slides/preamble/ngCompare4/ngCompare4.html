Preamble Angular VS AngularJS, what remains? How to port from 1 to 2?

To port from Angular1 to Angular, nothing magic, 3 important things:

  1. Prepare the existing application! Which means follow best practices, make it use recent technologies such as typescript/ES6, and use the angular router ported to angularJS
  2. Use angular1 directives, which will ease the process of converting angular1 directives to Angular components
  3. Include Angular in your AngularJS application, to make them coesxist together and convert things incrementally (more for developments, not sure using two frameworks at the same time in production is a good idea)
