Preamble Practical Work - Create your own Angular app

  • Let’s start writing our first Angular application! Open up the terminal and run the ng new command to create a new project from scratch: 
    $ ng new ng2ecom

  • angular-cli has a built in HTTP server that we can use to start our app.
    Back in the terminal, go to the root of our application 
    $ cd ng2eCom
    And try: 
    $ ng serve

    Your application is now running on localhost port 4200. Let’s open the browser and visit: http://localhost:4200
  • You can use the ng generate (or just ng g) command to generate Angular Components, Directives, Pipes and Services: $ ng g component bookDetails # will generate book-details.component $ ng g component books $ ng g component home $ ng g component contact $ ng g component basket $ ng g component profile $ ng g component login $ ng g interface ./beans/user $ ng g interface ./beans/comment $ ng g interface ./beans/book $ ng g interface ./beans/item $ ng g class ./beans/basket $ ng g service ./services/dataContainer # will generate data-container.service $ ng g service ./services/books $ ng g service ./services/contact $ ng g service ./services/localStorage $ ng g service ./services/news $ ng g service ./services/user $ ng g pipe ./pipes/filterField # will generate filter-field.pipe $ ng g pipe ./pipes/orderBy # will generate order-by.pipe $ ng g pipe ./pipes/updateData # will generate update-data.pipe
