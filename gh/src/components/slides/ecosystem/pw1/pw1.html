Preamble Practical Work - Create your own Angular2 app

  • Once you have Node.js setup, the next step is to install TypeScript. To install it, run the following npm command: 
     $ npm install -g typescript
  • Angular provides a utility to allow users to create and manage projects from the command line. It automates tasks like creating projects, adding new controllers, etc. It’s generally a good idea to use angular-cli as it will help create and maintain common patterns across our application. To install angular-cli, just run the following command: 
     $ npm install -g angular-cli
  • Now that you have your environment ready, let’s start writing our first Angular application! Let’s open up the terminal and run the ng new command to create a new project from scratch: 
     $ npm new ng2-eCom
  • angular-cli has a built in HTTP server that we can use to start our app. Back in the terminal, at the root of our application 
     
			$cd ng2-eCom 
    
			$ng serve

    Our application is now running on localhost port 4200. Let’s open the browser and visit: http://localhost:4200
  • Generating Components, Directives, Pipes and Services You can use the ng generate (or just ng g) command to generate Angular components: 
     
	$ ng g component books
    
	$ ng g component home
    
	$ ng g component contact
    
	$ ng g component bookDetails -> book-details.component
    
	$ ng g component basket
    
	$ ng g component profile
    
	$ ng g component login

    

	$ ng g interface ./beans/user
    
	$ ng g interface ./beans/comment
    
	$ ng g interface ./beans/book
    
	$ ng g interface ./beans/item

    

	$ ng g class basket + mv ./src/app/basket.ts ./src/app/beans/basket

    

	$ mkdir ./src/app/services
    
	$ ng g service ./services/books -> books.service
    
	$ ng g service ./services/contact -> contact.service
    
	$ ng g service ./services/dataContainer -> data-container.service
    
	$ ng g service ./services/localStorage -> local-storage.service
    
	$ ng g service ./services/news -> news.service
    
	$ ng g service ./services/user -> user.service

    

	$ mkdir ./src/app/pipes
    
	$ ng g pipe ./pipes/filterField -> filter-field.pipe
    
	$ ng g pipe ./pipes/orderBy -> order-by.pipe
    
	$ ng g pipe ./pipes/updateData -> update-data.pipe
