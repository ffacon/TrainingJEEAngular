Ecosystem ES6 Introduction - Arrow Functions

Arrow functions ( => ) are yet another feature which appeared on many languages such as scala, more recently java, and probably many more. But what does it add to Javascript, since anonymous functions can already be used for callbacks?

  • A lighter syntax
  • this is the one of the parent function it is called in!
class SampleThis{ constructor(){ this.set= [1,2,3]; this.value= 'Value'; } //No specific "this" is bound to arrow functions, // so this always refer to the same this than the method it is // used in testThis() { this.set.forEach( (id) => { if (this !== undefined){console.log('Arrow function: ' + this.value + ' '+ id);} else{console.log('Standard function: this is ' + this);} }); this.set.forEach( function(id){ if (this !== undefined){ console.log('Standard function: ' + this.value + ' '+ id);} else{console.log('Standard function: this is ' + this);} }); } } new SampleThis().testThis();
