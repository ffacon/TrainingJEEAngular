Ecosystem Typescript - Classes

We have seen that ES6 comes with classes. What does typescript add to this concept?

  • The declaration of properties (so array functions can be defined on a class as well)
  • The typing of functions

Below is the example seen in ES6

enum Gender {male, female}; class Person{ private gender: Gender; private firstName: string; private lastName: string; private marriedTo: Person; public friends: Set<Person>= new Set<Person>(); constructor(gender: Gender, firstName: string, lastName: string, private age: number){ this.gender= gender; this.firstName= firstName; this.lastName= lastName; this.age= age; } marryTo= (other: Person) : void => { this.marriedTo= other; } public addFriend(friend: Person) : void { if (!this.isFriend(friend)){ this.friends.add(friend); } } private isFriend(other: Person): boolean { return this.friends.has(other); } }
