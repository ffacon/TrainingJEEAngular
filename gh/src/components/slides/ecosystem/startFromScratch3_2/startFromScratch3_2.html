Ecosystem Start From Scratch - Install NVM

If you need to switch between different applications and projects on a regular basis, you may need to install NVM in order to run different versions of Node.


  • Installation is as simple as downloading the NVM for Windows installer from the latest release on GitHub. At the time of writing, 1.1.7 is the latest release.

  • Extact the Windows Installer from the nvm-setup.zip and launch it. (This may require a restart in order to use the nvm form commandline)

  • Use nvm list available to list the Node versions available to be installed

  • Use nvm install 12.13.1 to install the required Node version.

  • Use nvm use 12.13.1 to activate the version that was just installed

  • Use nvm List to confirme that the correct version of Node was installed and activated

See more details about usage of NVM tools.
