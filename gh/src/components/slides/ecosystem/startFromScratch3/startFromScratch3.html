Ecosystem Start From Scratch - transpilation

In order to run your app correctly, your browser needs an ES5 version of your app. So, what are your choices if you use typescript or ES6.

If you use typescript, you can decide to transpile your code directly into ES5 which is perfectly fine. You can do that by different ways:

  • With tsc: the official typescript compiler, which is the usual way to compile typescript
  • With typescript.js which will do the transpilation at runtime. This is really interesting to POC something directly on the web (this way, you don't need any tool, everything is done by the browser). It's bad for performance purpose and won't let you check for compilations errors before runtime...
