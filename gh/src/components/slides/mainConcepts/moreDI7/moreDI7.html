Main Concepts Dependency Injection - More Details - Bootstrap

The bootstrap allows you to provide services which will then be available everywhere in the app ( or a new instance of them if you redefine a new provider through inheritance )

import {Component} from '@angular/core'; import {bootstrap} from 'angular2/platform/browser'; import {Person} from 'services/Person'; @Component({ selector: 'main-cmp', template: 'Hello World' }) class MainCmp{ constructor(public person: Person){} } bootstrap(App, [Person]);
