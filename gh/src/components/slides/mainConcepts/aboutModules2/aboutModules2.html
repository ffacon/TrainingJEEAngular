Main Concepts About Modules - Services and Pipes

  • To take benefit of DI, you have to declare the Services or Pipes you want to inject ( don't mix up inject a pipe into a component and use a pipe into a template )
  • This is done through the providers field (and can also be done at the component level if you want to limit their accessibility to a subpart of the tree)
  • @Injectable class CustomService {} @Pipe({name:'customPipe'}) class CustomPipe{} @NgModule({ imports: [BrowserModule], declarations: [AppComponent], providers: [CustomService, CustomPipe], bootstrap: [ AppComponent ] }) class AppModule { } platformBrowserDynamic().bootstrapModule(AppModule);
