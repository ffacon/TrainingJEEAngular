Main Concepts Dependency Injection - More Details - Provide

provide will be usefull typically when you will want to change an implementation.

In the sample below, inject a Person somewhere will result in retrieving a PersonImpl instance

import {ReflectiveInjector} from '@angular/core'; //Define our injector and its providers let injector = ReflectiveInjector.resolveAndCreate([ { provide: Person, useClass: PersonImpl}, { provide: LoggerService, useClass: LoggerService} ]); // Then ask our injector to use its providers // to get an instance of Person let person:Person= injector.get(Person);
