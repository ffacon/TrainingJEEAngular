Main Concepts About Modules - Components and pipes into templates

  • Components and Pipes are generally used in templates, and to use your custom ones, you need to declare them on the module: @Pipe({name:'customPipe'}) class CustomPipe{ //... } @Component({ selector:'cmp2', template:`{{'Sub component' | customPipe}}` }) class Cmp2{} @Component({ selector:'app-cmp', template:` <h1>Hello</h1> <cmp2></cmp2> ` }) class AppComponent{} @NgModule({ imports: [BrowserModule], declarations: [AppComponent, Cmp2, CustomPipe], bootstrap: [ AppComponent ] }) class AppModule { } platformBrowserDynamic().bootstrapModule(AppModule);
