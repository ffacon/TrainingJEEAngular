Main Concepts Dependency Injection - More Details - Injector

The example below show you how to use programmaticaly an injector to

  • Declare providers, which in fact are a { key:value } pair
  • And associate them to the injector
  • Before using them
import {ReflectiveInjector} from '@angular/core'; //Define our injector and its providers let injector = ReflectiveInjector.resolveAndCreate([ {provide: Person, useClass: Person}, {provide: LoggerService, useClass: LoggerService} ]); // Then ask our injector to use its providers // to get an instance of Person injector.get(Person); import {ReflectiveInjector} from '@angular/core'; //Define our injector and its providers let injector = ReflectiveInjector.resolveAndCreate([ Person, LoggerService ]); // Then ask our injector to use its providers // to get an instance of Person injector.get(Person);

As in the last tab, if the key and the type are the sames, you don't need to use the provide function, instead use the type only is a shortcut.
