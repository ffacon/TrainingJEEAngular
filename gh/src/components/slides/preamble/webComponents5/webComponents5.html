Preamble About Web Components - Semantic

  • In order to make a website accessible, you have to think about semantic tags
  • <div> and <span> are not semantic whereas <article> and <footer> are
  • But what about <myCustomWebComponent> ?
  • Prefer using understandable element names
  • Consider using microformats
  • Microformats allow to add metadata to your page using standard attributes such as class, rel or rev
<book title="the title" classification="fiction" author="the author" price="12€" ></book> <article> <h1 class="title">the title</h1> <ul> <li class="author">the author</li> <li class="classification">fiction</li> <li class="price">12€</li> </ul> </article>
