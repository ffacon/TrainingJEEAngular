Advanced Concepts e2e testing

  • End to End testing is done in Angular with protractor.
  • Use npm to install Protractor globally with:
    • npm install -g protractor
  • The protractor install also provides you the webdriver-manager executable
  • webdriver-manager allows you to get and manage the selenium server.
    Use it to download the necessary binaries to let selenium drive your browser during e2e testing:
    • webdriver-manager update
